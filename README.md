## Database

Postgres on [Neon](https://neon.tech), accessed through [Drizzle ORM](https://orm.drizzle.team).

1. Create a Neon project, copy its connection string, and put it in `.env.local`:

   ```
   DATABASE_URL=postgresql://user:password@host/dbname?sslmode=require
   ```

2. Schema lives in `src/db/schema.ts`. Migrations are SQL files under `drizzle/`,
   generated by diffing the schema against the prior migration.

   ```bash
   # After editing src/db/schema.ts:
   npm run db:generate   # writes a new SQL file under drizzle/
   npm run db:migrate    # applies pending migrations to DATABASE_URL
   npm run db:studio     # open Drizzle Studio in a browser
   ```

   Migrations are checked in. Review the generated SQL before running
   `db:migrate`, especially against shared environments.
